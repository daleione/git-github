# git-github

## Installation

```shell
cargo install git-github
```

## How to use

1. cd into the git repository

```shell
  git github open
```

## Sub command open

```shell
  Open the repo website in your browser

  Usage: git-github open [OPTIONS]

  Options:
    -c, --commit <COMMIT>
    -b, --branch <BRANCH>
    -r, --remote <REMOTE>  [default: origin]
    -h, --help             Print help
```
